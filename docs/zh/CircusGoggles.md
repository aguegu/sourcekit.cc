# 概述

Sourcekit&reg; Circus Goggles 是一款具有 32 像素的 RGB LED 护目镜。只需转动板上的 5 个旋钮，即可带来绚丽的动画效果。

# 安全注意事项

* 远离儿童。儿童使用 Circus Goggles 需要成人指导。
* 灯光亮起时不要直视 LED。可能会伤害您的视力。
* 在干燥的地方存放或使用 Circus Goggles，不要让它接触水、雨或雪。
* 如果 Circus Goggles 接触水，请将其放在室温下的干燥位置，放在柔软的吸水毛巾上，让其彻底干燥。在完全无水分之前不要操作设备。
* 仅使用适当的 USB 电源插座或 PC USB 面板为 Circus Goggles 充电。

# 特性

* 32 个 WS2812 RGB LED 排列成 2 个完整的圆圈
* 5 个旋钮可调节动画效果
* 主控制器为 [STC](http://stcmcudata.com/)8H8K64U 微控制器
* 由可充电 3.7v/180mAh 锂离子电池供电，具有短路/过载/过充/过放保护
* 通过标准 5V 1A micro USB 插座充电或固件升级
* 电池电压指示器
* 电池续航 1-6 小时，具体取决于效果
* 路径切换开关
* 麦克风输入作为变体
* 电池充电指示灯 LED
* 可在线升级固件

# 使用说明

* 打开电源开关。如果没有 LED 亮起，电池可能已耗尽，请使用 micro USB 电缆充电一段时间。通过 USB 电缆供电时，一些 LED 应该会亮起。电池同时正在充电。
* 调节 5 个旋钮：部分/颜色/亮度/速度/图案，然后按下路径切换按钮选择您喜欢的效果。

# 调节指南

通过旋钮和路径的所有这些组合，Sourcekit&reg; 提供了数千种效果，无需第三方帮助，如遥控器、PC 或移动应用程序。受电子乐器的启发，Sourcekit 设计师希望 Circus Goggles 成为一款具有简单和经济设计的独立产品。随着入门门槛的降低，孩子们可以轻松享受乐趣，并自行探索 LED 运动或颜色图案。

当 LED 太亮时调暗，或者当闪烁太快可能伤害眼睛时减慢速度。

颜色/亮度和速度旋钮直接工作。

`图案旋钮`在当前固件中有 4 种图案类型。

1. 单点：颜色会根据板载麦克风拾取的环境声音水平略有变化。
2. 彗星：尾部的长度可通过亮度旋钮控制。
3. 全色彩空间
4. 半色彩空间

`部分旋钮`有 6 个可能的值：1、2、4、8、16、32。这意味着`图案`将有多少个副本投影到 LED 上。例如，当图案为单点且部分为 1 时，您只会看到一个移动的 LED 亮起。如果部分为 8，您将看到 8 个相同颜色的 LED 移动。因此，当部分处于最大值 32（这也是 LED 总数）时，整个护目镜将以相同颜色亮起。但同时，颜色旋钮也是可调的。

`路径选择按钮`在当前固件中有 4 条路径。`路径`是这 32 个 LED 按顺序组织的方式。它会使动画看起来与默认路径完全不同。

# 功耗

按下电池检查按钮，Circus Goggles 会显示电池的 3 位电压表（大致正确），通过它可以计算剩余电池寿命。

电池电压 = (红色 LED 数量) * 1 + (绿色 LED 数量) * 0.1 + (蓝色 LED 数量) * 0.01（单位：伏特）

充满电时电压约为 4.20v，停止放电时为 3.40v。

在电池容量恒定的情况下，电池寿命取决于 LED 效果的安排方式。因此，亮起的 LED 越少，颜色越接近原色（红、绿、蓝），所需的电流就越少。

主控制器板旁边还有一个电池充电指示灯 LED（橙色）。电池正在充电时会亮起，电池充满且充电完成时会熄灭。
